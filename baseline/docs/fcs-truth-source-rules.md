# FCS 统一事实源冻结规则

适用范围：
- 统一数据源 / 统一事实源
- 兼容层收口
- 页面仅做字段绑定切换

## 允许修改
- 数据源
- 映射层
- 查询层
- 状态来源
- 兼容层
- 旧字段到新字段绑定

## 禁止修改
- 页面布局
- 交互路径
- tab 结构
- 表格列结构（除非业务任务明确要求）
- 筛选器结构
- 统计卡结构
- drawer / modal 交互
- 页面信息层级
- 默认详情展示方式

## 执行约束
- 统一事实源类任务中，禁止顺手改 UI 与交互。
- 若确需修改 UI/交互，必须由单独任务明确提出，且与“数据源整改”拆分执行。
- 兼容层可以保留旧 shape，但不得继续作为主真相源。
